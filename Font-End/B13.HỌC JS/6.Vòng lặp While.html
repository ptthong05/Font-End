Vòng lặp while luôn luôn kiểm tra điều kiện trước, nếu điều kiện thỏa mãn thì mới chạy vào trong vòng lặp.
Cú pháp:
    while (condition) {
    // code
}
Trong đó:
    condition: là điều kiện thực hiện vòng lặp.
    hi condition có giá trị true thì code được thực thi. Ngược lại, khi condition là false thì vòng lặp kết thúc.






Khác với vòng lặp while, vòng lặp do...while luôn thực hiện ít nhất một lượt lặp. Sau đó mới kiểm tra điều kiện lặp.
Cú pháp:
    do {
    // code
    } while (condition);
Trong đó:
    condition: là điều kiện thực hiện vòng lặp.
    Khi condition có giá trị true thì code được thực thi. Ngược lại, khi condition là false thì vòng lặp kết thúc.
