Hoisting

    Biến trong hoisting
Hoisting nghĩa là có thể sử dụng 1 biến xong sau đó mới cần khai báo biến đó.
Javascript sẽ di chuyển toàn bộ các khai báo biến lên đầu chương trình. Vì vậy, những dòng code có sử dụng biến mà chưa khai báo sẽ không bị lỗi.



    Từ khóa let, const và var trong hoisting
Sử dụng var:  Nếu biến chưa gán giá trị thì sẽ trả về undefined.
Sử dụng let và const: Nếu biến chưa gán giá trị thì sẽ xuất hiện thông báo lỗi a is not defined. Lỗi này muốn nói rằng biến a chưa được định nghĩa.



    Hàm trong hoisting
Declaration Function
    Có tính hosting
    Cú pháp: 
            function tenHam() 
                { // Code }

Expression Function
    Không có tính hosting
    Cú pháp: 
            var tenBien = function() 
                { // Code }

Arrow Function
    Không có tính hosting
            var tenBien = () => 
                { // Code }
